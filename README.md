# mango-leaf-disease-prediction
machine learning project for detecting and predicting diseases in mango leaves using image processing and classification techniques
Mango Leaf Disease Prediction

Project Overview

This project uses Machine Learning and Image Processing techniques to detect and predict diseases in mango leaves from images.

Features

- Disease detection from mango leaf images
- Image preprocessing
- Machine Learning based classification
- Accurate disease prediction

Technologies Used

- Python
- Machine Learning
- Image Processing
- OpenCV
- NumPy
- Scikit-learn

Objective

To help farmers identify mango leaf diseases at an early stage and improve crop health management.
